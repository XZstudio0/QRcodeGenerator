# QRcodeGenerator
script to create QR code in python

1. Download the required libraries:
pip install qrcode pip install barcode

2. Then download the repository with:
 git clone [https://github.com/XZstudio0/QRcodeGenerator](https://github.com/XZstudio0/QRcodeGenerator.git)

3.Go to the directory with the script by writing: 
cd QRcodeGenerator

4.Run the script: 
python YouTubeDownloader.py
